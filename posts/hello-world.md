# 你好，世界！

欢迎来到我的博客！这是第一篇示例文章。

## 快速开始

这个博客系统非常简单易用：

1. **写文章**：在 `posts/` 目录下创建 `.md` 文件
2. **配配置**：在 `config.json` 中配置文章信息与可见性
3. **跑起来**：`npm start` 启动服务器

## Markdown 功能展示

### 代码块

```javascript
function greet(name) {
  console.log(`你好，${name}！`);
}

greet("世界");
```

## 关于

这是一个由 **Node.js + Express + Marked** 驱动的简约博客系统。所有内容都基于文件系统，无需数据库，方便管理和部署。
